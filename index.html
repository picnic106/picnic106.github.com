关于穷人要如何成功

发表于 2015-09-03   |   分类于 生命哲思

作者：清云大师
摘自:【成功哲思】關於窮人要如何成功

詢問我關於窮人要如何成功的命題,太多了。

  1. 依靠格言註定失敗,因為,成功從來不會寫入格言之中。拋棄格言,必須讀整本書,一本又一本的讀,絕對不能讀重點。

  2. 一位高爾夫選手要取得金牌,是要走過幾萬公里路程的生住異滅與觀照時光,而走過幾萬公里路程的人,卻未必能取得金牌。我的小叔在大陸是短跑名將,由此得到一命,不然在1950年代已被槍斃了。迄今八十多歲,健步如飛,只是個性詭異,他訓練選手是一直跑,跑到死亦不可放棄對跑步的熱情。

  3. 我總是說,按照自己興趣,做就對了,絕對不要退縮,亦不可空談理論。臉書上的格言是消遣的,娛樂的,沒有可思考的內涵。

  4. 所有關於成功的書籍,通通無效。因為,成功無以寫入書中。

  5. 行萬里路是最重要的,內涵即是一門深入,永不退縮,絕不空談。

如果換成是你,你願意對一個毫無希望的窮困項目繼續堅持十年嗎?

1,成功绝不会在书里传授给你，一是因为作者不可能把自己辛辛苦苦获得的成功经验告诉你的，最伟大的厨师在收徒的时候，总会留一手。二是因为书这个载体传授不了成功，成功要靠你不停不停的积累，做一件事情，你不停的做一万遍，也许你接近了成功。

2,成功在于一门深入，不停的深入，即使是机械化的做，直到某一刻你会领悟。就像我学游泳，刚开始学蛙泳总是很费力，后来不停的游，不停的游，突然领悟到，在蛙泳，手脚伸直的时候，可以漂浮啊，可以休息啊，从此，蛙泳游3公里不是梦。做一件事，不要怕困难，不停的做，不停的做，永不退缩，你总会领悟。

你为何还没有成功？

发表于 2015-09-02   |   分类于 生命哲思

我看过一个世外高人说：

走过童年，长大后突然要学古文，行吗？你应该静心回忆，已一路的成长历程，兴趣是什么？花了最多时间的专业是什么？所谓鑑往知来，你不了解过去吗？
迎合别人注定是失败，你得找到你的本质，一门深入，乃发光发热。
如果你小时候喜欢看煮菜电视，自己动手煮菜，却是一路文学，到了30岁没成就，那么不妨回去煮菜，从一张白纸的学徒做看看，或许会激发童年基因，一个潜在能量的特大爆发。
有些人是中年时期投入小发明的发明大王，概是源于童年时期喜欢动手发明，却一路读了文学和艺术，糊涂了40岁，突然来了个善缘，又动手发明，椅子、桌子、筷子，灵感来了，童年基因大爆发，十年间申请的专利吓死人。

看到这句话，我开始思考自己的人生，为何到了而立之年还是一事无成。我的生活分为三个阶段，小时候生活在农村由爷爷奶奶带大；到了小学四年级离开爷爷奶奶生活到了城市；然后读完高中培训完，离开爸妈，来到北京生活，组成家庭。

我从小生活在爷爷奶奶身边，我爷爷是在我们村的法师，哪家有怪事，例如晚上窗户老响，例如某家的鸡一直呆在窗户上，我爷爷就去给他们做法，首先就是要有肉，米，然后我爷爷用一些手势和脚法，嘴里念念有词，有一套法师做法流程，而一般结果都很好，做完法师，怪事一般就会消失。我还记得有一次有家的牛不见了，我爷爷就开始用手指计算，然后就说牛在哪里，果然牛就在那里，我想手指计算只是障眼法。而且我爷爷是村里日常琐事的调解员，哪家的鸡怎么了，哪家的牛吃了谁家的作物，都会去找我爷爷评评理。爷爷的脾气特火爆，只要我做错一点事就会被骂，而且特别难听，所以从小没少挨过爷爷骂。反正爷爷就是一个法师，在我们村特别受人尊敬，而且是辈份最高的一位。而我的奶奶就主管家庭，种菜，收菜，做饭。我印象最深的就是跟奶奶去摘茶叶，然后把茶叶捣鼓晒干。从我来北京，就再也没有喝过那么纯正的茶叶，只有我奶奶做的茶叶最好喝，因为特别干净，别家的里面不是有杂草，就是茶叶大小不一。真的怀念。

在农村的阶段，现在回忆起来，我也不知道我喜欢什么。就是喜欢玩吧。要说喜欢的就是奶奶的茶叶，躺在门口树上乘凉。从这点来说，我想农村的孩子就比城市的孩子要落后很多。如果穷人要成功，必须要从小学起，不到一岁或者三四岁就要开始培训，养成某种专业的基因，这样才有可能成功，例如体操基因，例如文学基因，例如画画基因。现在的城市学生，从小学钢琴，学各种课外辅导班，就可以打败农村人。除非，有好的家庭教育，从小就让小孩读书，不去玩，那么才有可能成功，例如我一个亲戚，读书特别厉害。我想我就是属于那种大部分农村人，小时候不知道要干什么，就是随着爷爷奶奶干农活，也不知道自己喜欢什么。但是我想农作物的生活习性也许已经植入我的脑海里，例如花生种植，玉米种植，水稻种植，烟草种植，这只我从小耳濡目染的。如果我现在去做农作物种植，也许会成功。

想到这里给我一些教育孩子的启发，现在社会所深恶痛绝的从小就逼小孩学各种事物，其实对小孩子来说是好的，并不是一无是处。你学了，才知道你要什么东西，你知道你喜欢什么东西，然后你再一门深入，长大了遇到善缘一定会成功。所以我小孩喜欢画画，他来北京一定要给他报一个画画班，喜欢读书，一定要让他多读书，英语班是一定要报的，其他再看看。

我记得我小时候不喜欢上学，上一年级的时候，自己偷偷跑回家，然后又被妈妈打着去上学，反正对上学特别抗拒。我上小学四年级的时候，被爸爸妈妈接到城市生活。我爸爸是一个环卫工人，每天早出晚归，而且每次回来都是一身臭汗味。我妈妈也跟着爸爸一块做。虽然做环卫工人工资低，但是靠着一些废品还是把我和妹妹养大。在班上我的性格就是不喜欢说话，内向，在人多的讲台说话，都会心跳加速，跟喜欢的女孩子说话都会心跳加速。但是我记得我喜欢写作文，喜欢看作文书，而且我写作文的时候，老是喜欢以一种很平静的语气写作，似乎想用风轻云淡的风格写作。但是都没有成功过，因为我写作也不是班上最好的，成绩也不是班上最好的。我记得我最喜欢的事情就是放学后和同学一起打篮球。

回忆到这里，突然觉得，很悲观，我到底喜欢什么呢？农作物种植可以一试，也许会成功。

小森林-冬春篇:满满都是回忆

发表于 2015-09-01   |   分类于 电影

去年看完小森林-夏秋篇，今天乘着有点空闲，把小森林-冬春篇看完了，这里是观看地址。

高中时候喜欢看严井俊二的电影，记得有一部关于莉莉周的一切,里面很多镜头是主角市原隼人

穿梭在城市与农村之间，日本片又很擅长长镜头，当时大片的稻田给我很深的印象，虽然当时这部片不是以农村为主的题材，但是让我经常回忆起小时候和爷爷放牛、收稻谷、随奶奶摘茶叶的事情，现在他们都已不在，却是我最宝贵的精神财富。所以之后就一直喜欢看日本关于描述农村的电影，例如哪啊哪啊神去村。

而这两部小森林却让我想起生活在城市的时候，曾经跟自己说，有一天我要回农村，自己要掌握农作物的生长季节以及生活习性，何时种黄瓜，何时种冬瓜，何时种白菜，何时种水稻，何时收割，有杂草要怎么办，花生要怎么打理。而这部小森林就是这样一部电影，主题是女主角桥本爱逃避生活一个人生活在农村，妈妈不知去向，一个人种植农作物，孤独的跟着每种农作物的习性而生活，然后把它们做成美食，从中穿插主角对人生的一些思考和回忆。

其中也对日本农村现代化表示敬佩，可能是日本人的严谨。厨房各种道具齐全，腌菜坛子，各种菜刀，各种碗，擀面杖，微波炉，冰箱等，井井有条，随你要做什么美食，各种道具都有。 对女主角由农作物做成的美食钦佩不已，印象深刻的是，用红豆和南瓜做蛋糕。影片从红豆的种植，什么季节种，什么时候除杂草，什么时候适合收割都说得一清二楚，对红豆的种植习性了解得一清二楚。导演一定是农村出生的，要不然拍不出如此精确的电影。这电影不是走过场，而是导演在认真的拍，春夏秋冬四个季节的景色，每种农作物的习性，如何存储，如何做成美食，事无巨细。就是一部农村百科全书。
我想这部电影不适合喜欢爆米花电影的人群，而适合生活在农村，憧憬在农村生活的人群。特别是现在中国社会浮躁，大批人群往城市拥挤，但是农村却开始出现断代了，因为已经没有了年轻人生活在农村。也许这是社会成长的代价。

已而立之年，却一事无成，是在这里抱怨，然后也逃避回农村躲闪吗，但是你有家庭责任，你还要奋斗。但是无论怎样，我的童年是在农村度过的，随着爷爷放牛，跟着奶奶摘茶叶，种菜，扯花生。农村是我的故乡，我爱那里的山，那里的水，那里的风，那里的稻田。

python批量修改文件

通过一天的学习发现python相对于java有几大不同点：

  1. 变量不用申明什么类型，java是一定要申明的;

  3. 最大的不同python是脚本语言，不用编译再运行，直接运行即可；
  4. 代码相较于java要简洁许多，例如打开文件流，java至少四五行，而python一行就行:open(fileObj,'r+',encoding= 'utf-8')。

不要抱怨、不要慵懒、不要三心二意，这是通向成功之钥，也是人与人不同的最根本的原因。

干什么事就好好干，不要抱怨。如果你不想干活，在听音乐就好好听音乐，听出名堂。不要在听音乐的时候还在抱怨我没干活，到时候两件事情都干不好。
